Latest Release Candidate: Shoal GMS for Glassfish 3.1

Free Production-Quality Java based Dynamic Clustering Framework

Shoal is a free, open-source, production-ready, supported clustering framework based on Java. The framework provides a foundation for building fault tolerance solutions through a cluster lifecycle event model and messaging bus. Shoal is the runtime clustering engine for GlassFish 3.1. 

The Shoal GMS 1.5.29 release provides all of the capabilities of the 1.1 release, plus many new improvements such as pluggable transport, reliable delivery of Master Change Events to the group and much more. The tested and supported pluggable transport is Grizzly 1.9.28. The jxta 2.5 transport is still available via a source code build; however, since it is not tested, it is not included in pre-built shoal-gms.jar.

Final Release: Shoal 1.1 FCS

Free Production-Quality Java based Dynamic Clustering Framework

Shoal is a free, open-source, production-ready, supported clustering framework based on Java. The framework provides a foundation for building fault tolerance solutions through a cluster lifecycle event model and messaging bus. Shoal is the runtime clustering engine for GlassFish,  and Sailfin (Telecom) application servers. 

The Shoal 1.1 release provides all of the capabilities of the 1.0 release, plus many new improvements such as JoinedAndReadyNotificationSignal to allow members to notify group members when they are ready to process operations, ability to specify particular network interface on which to run Shoal/Jxta communications, ability to join multiple clusters by the same member, cross subnet clustering support(Experimental), reporting failures during network and hardware failures, new notifications for GroupLeader change, and much more.
