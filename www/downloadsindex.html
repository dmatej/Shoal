Latest Final Release: Shoal 1.0 Update Release

Free Production-Quality Java based Dynamic Clustering Framework

Shoal is a free, open-source, production-ready, supported clustering framework based on Java. The framework provides a foundation for building fault tolerance solutions through a cluster lifecycle event model and messaging bus.

Work in Progress: Shoal 1.1

The Shoal 1.1 release provides all of the capabilities of the 1.0 update release, plus many new improvements such as a JoinedAndReadyNotificationSignal, ability to specify particular network interface to run Shoal/Jxta communications on, ability to join multiple clusters by the same member, cross subnet clustering support, reporting failures during network and hardware failures, and much more.

Shoal 1.1 is work in progress.
