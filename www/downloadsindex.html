Latest Stable Release: Shoal 1.0 Update Release

Free Production-Quality Java based Dynamic Clustering Framework

Shoal is a free, open-source, production-ready, supported clustering framework based on Java. The framework provides a foundation for building fault tolerance solutions through a cluster lifecycle event model and messaging bus.

The Shoal 1.1 release provides all of the capabilities of the 1.0 update release, plus many new improvements in the developmental stage such as a JoinedAndReadyNotificationSignal, ability to specify particular network interface to run Shoal/Jxta communications on, ability to join multiple clusters by the same member, cross subnet clustering support, and much more.
